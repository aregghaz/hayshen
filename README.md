# HayShen

Чистая основа React-проекта с JSX.

Алиас `@` указывает на папку `src`:

```jsx
import Header from '@/components/Header/Header';
```

## Команды

- `npm start` — запустить локальную разработку
- `npm run build` — собрать production-версию

## Структура

```text
src/
├── App.jsx
├── App.scss
├── index.jsx
└── index.scss
```

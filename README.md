# HayShen

React նախագիծ՝ կառուցված JSX-ով։

`@` կեղծանունը հղվում է `src` թղթապանակին։

```jsx
import Header from '@/components/Header/Header';
```

## Հրամաններ

- `npm start` — գործարկել տեղային մշակման միջավայրը
- `npm run build` — ստեղծել production տարբերակը

## Կառուցվածք

```text
src/
├── App.jsx
├── App.scss
├── index.jsx
└── index.scss
```
